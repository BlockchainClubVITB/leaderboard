# React Leaderboard App

A modern web application built with React, TypeScript, and Vite that displays and manages leaderboard data.

## 🚀 Tech Stack

- React 18
- TypeScript
- Vite
- ESLint
- SWC/Babel

## 🛠️ Getting Started

1. Clone the repository
2. Install dependencies:
```bash
npm install
```
3. Start the development server:
```bash
npm run dev
```
